## Origin types
- `IN-HOUSE` — built from scratch in this project (Blender, Blueprint, etc.)
- `OPEN-SOURCE` — sourced externally; must include source URL and license

## Rules
- Update this file whenever an asset is added or changed.
- Open-source assets must always keep attribution and license.
- In-house assets can carry a maker's mark in metadata or in-world if it fits the tone.
